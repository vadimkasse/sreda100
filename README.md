# SREDA100 (v20.4)

Генератор акцидентной типографики с геометрическими искажениями и хроматической аберрацией.

## Что нового в v20.4:
- **Чистая Геометрия**: Удалены все "жидкие" эффекты (chaos, twist, ripple). Теперь всё строится на жестких разрезах и сдвигах.
- **Новые Эффекты**:
    - `slices`: Диагональная нарезка изображения.
    - `blocks`: Прямоугольные цифровые сдвиги (глитч).
    - `prism`: Кристаллические переломы по всему холсту.
    - `columns`: Вертикальные ритмичные сдвиги.
- **Хроматическая Аберрация**: Независимый случайный сдвиг красного и синего каналов (диапазон 18-54px на 2x рендере).
- **Композиция**: Переход от паттерна к **одному крупному слову** по центру экрана.
- **Типографика**: Максимальный размер шрифта увеличен до 400 для массивного вида.

## Технический стек:
- Python 3 + Pillow + NumPy.
- Суперсэмплинг 2x (рендеринг в 4K -> даунскейл до 1080p для сглаживания).
- Билинейная интерполяция при деформациях.

## Как использовать:
```bash
python3 sreda100.py --batch 20 --day FRIDAY
```
Результаты сохраняются в `~/sreda100_output`.
