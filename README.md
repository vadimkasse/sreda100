# SREDA100 (v20.4)

**SREDA100** — это авторский арт-проект, исследующий границы акцидентной типографики через алгоритмические искажения. Это серия визуальных экспериментов, где текст становится архитектурным объектом, подвергаясь цифровой деконструкции.

Проект представляет собой кастомный пайплайн, в котором классические шрифтовые формы сталкиваются с агрессивной геометрией и оптическими аберрациями.

## Концепция и Визуальный язык (v20.4):
- **Геометрическая Деконструкция**: Отказ от плавных "жидких" форм в пользу жестких разрезов. Текст дробится на слайсы, блоки и кристаллические грани.
- **Цифровой Шум**: Использование эффектов `slices` (диагональная нарезка), `blocks` (прямоугольный глитч) и `prism` (многогранные переломы).
- **Оптические Искажения**: Мощная хроматическая аберрация с независимым вектором сдвига для каждого цветового канала, имитирующая физические дефекты линз или сбои видеосигнала.
- **Монолитность**: Акцент на одном массивном слове (размер шрифта до 400), занимающем центральное место в композиции.

## Техническая реализация (Case Study):
Пайплайн спроектирован для высокопроизводительной генерации в облачной среде **Modal**, что позволяет мгновенно создавать тысячи уникальных итераций одного образа.

- **Рендеринг**: 2x суперсэмплинг (генерация в 4K -> даунскейл до 1080p через фильтр LANCZOS для сохранения четкости граней).
- **Математика**: Деформации реализованы через NumPy (билинейная интерполяция карт смещения).
- **Инфраструктура**: Масштабируемый запуск через `modal_app.py`.
