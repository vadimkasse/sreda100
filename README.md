# SREDA100 (v20.5)

**SREDA100** — это авторский арт-проект, исследующий границы акцидентной типографики через алгоритмические искажения. Это серия визуальных экспериментов, где текст становится архитектурным объектом, подвергаясь цифровой деконструкции.

Проект представляет собой единый пайплайн для генерации статических изображений и ритмичных видео-манифестов.

## Концепция и Визуальный язык (v20.5):
- **Геометрическая Деконструкция**: Отказ от плавных "жидких" форм в пользу жестких разрезов. Текст дробится на слайсы, блоки, колонны и кристаллические грани.
- **Двойной Режим**:
    - **Static**: Создание уникальных застывших состояний с глубокой детализацией. Одно крупное центрированное слово (fs до 400).
    - **Video (Hold & Burst)**: Ритмичная анимация (10 FPS), чередующая фазы покоя (Hold) и агрессивные глитч-взрывы (Burst).
- **Оптические Искажения**: Мощная хроматическая аберрация с независимым вектором сдвига для каждого цветового канала (18-54px). В видео-режиме аберрация "дышит" в ритме глитча.

## Техническая реализация (Unified Engine):
Пайплайн спроектирован для высокопроизводительной генерации в облачной среде **Modal**.

- **Рендеринг**: 2x суперсэмплинг (генерация в 4K -> даунскейл до 1080p через фильтр LANCZOS).
- **Видео-движок**: Генерация последовательности кадров (10 FPS) с использованием стейт-машины для управления ритмом и интерполяцией.
- **Инфраструктура**: Деплой через `modal_app.py`, хранение в Cloudflare R2.

## Использование и Деплой:

**Деплой в Modal (Cloud API):**
```bash
modal deploy modal_app.py
```
После деплоя API доступен по адресу: `https://kassevadim--sreda100-generate-endpoint.modal.run` (POST запрос с JSON `{"day": "FRIDAY"}`).

**Локальная генерация артов:**
```bash
python3 sreda100.py --batch 20
```

**Локальная генерация видео (5 сек, 10 FPS):**
```bash
python3 sreda100.py --batch 5 --video
```
